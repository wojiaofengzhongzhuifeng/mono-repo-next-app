Common Tailwind 插件演示

展示各种插件的功能和用法

Center Container 演示

这个容器使用了 .center-container 类， 它会自动居中并具有响应式的内边距和最大宽度。

响应式内边距

• 默认: px-3 (12px)

• 375.1px+: px-4 (16px)

• 768.1px+: px-6 (24px)

响应式最大宽度

• 默认: calc(100vw - 24px)

• 375.1px+: 736px

• 768.1px+: 1200px

其他变体

• center-container-sm

• center-container-md

• center-container-lg

• center-container-xl

Spacing 插件演示

基于 2 倍规则的间距系统：数字 n 表示 n*2px 的间距

示例间距

p-10 = 20px
p-20 = 40px
p-30 = 60px

Margin

m-2 = 4px
m-4 = 8px
m-6 = 12px

Gap

范围

• 0-99

• 0 = 0px

• 99 = 198px

• px = 1px

Font Size 插件演示

支持 12px 到 80px 的字体大小，以及字体大小/行高的组合

基础字体大小

font-12: 12px 字体大小

font-16: 16px 字体大小

font-20: 20px 字体大小

font-24: 24px 字体大小

font-32: 32px 字体大小

字体大小/行高组合

text-12/14: 12px 字体, 14px 行高

text-16/20: 16px 字体, 20px 行高

text-20/28: 20px 字体, 28px 行高

text-24/32: 24px 字体, 32px 行高

text-32/40: 32px 字体, 40px 行高

Section Block 插件演示

内容区块间距插件，支持响应式和不对称间距

对称间距示例

section-block-20

上下间距: 40px (20*2)

section-block-30

上下间距: 60px (30*2)

section-block-45

上下间距: 90px (45*2)

不对称间距示例

section-block-30-20

上间距: 60px, 下间距: 40px

section-block-45-20

桌面端: 90px, 移动端: 40px

section-block-top-20

仅上间距: 40px
